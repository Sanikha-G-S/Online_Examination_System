# Online Examination System

A web-based examination platform developed using Python, Streamlit, and SQLite.

## Features

- User Registration & Login
- Role-Based Access Control
- Admin Dashboard
- Exam Creation
- Question Management
- Automatic Evaluation
- Student Results
- Analytics Dashboard
- Leaderboard
- PDF Result Reports

## Technologies Used

- Python
- Streamlit
- SQLite
- Pandas
- ReportLab

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
